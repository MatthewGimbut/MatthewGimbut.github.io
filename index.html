Message Encoder

Description

This project was a simple message encoder created to allow friends to communicate in secret messages. It takes a string entered by the user and converts it to a binary representation of each character's ASCII value. The 1s and 0s are then replaced by strings entered by the user. It can also be used to decode messages as well. Messages can be send to others over gmail using the input fields.

Technologies

C#, Visual Studio

  • Date: September 2016

Acme Airlines

Description

This project was created for my Data Abstraction class in Fall 2015. It is meant to imitate software that an actual airline would use. The program keeps track of all listed planes and displays the their routes including time of departure and time of arrival. It also keeps track of individual accounts, allows for reservation and cancellation of seats. The project is fully serialized so progress is stored between uses.

Features

Functional email/password validation

Functional registration/login system

Dynamic flight display

Multiple tiers of user accounts

Technologies

Java, Swing

  • GitHub: Repository
  • Date: Fall 2015
